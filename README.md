# TerM Peatland Carbon Module — Interactive Demo

ENVIROMIS-2026 · Poster supplement

## Pages
- **index.html** — Main landing page
- **heterotrophic.html** — Interactive heterotrophic respiration model
- **vegetation.html** — GPP experiment explorer (15 runs)

## Deploy to GitHub Pages
1. Create new repository on GitHub
2. Push this folder: `git push origin main`
3. Settings → Pages → Source: GitHub Actions
4. URL will be: `https://USERNAME.github.io/REPO/`

## QR codes for poster
Generate QR codes pointing to:
- `https://USERNAME.github.io/REPO/heterotrophic.html`
- `https://USERNAME.github.io/REPO/vegetation.html`
